# Project Timber
Timbr is a web service/application that turns an arbitrary web page into structure JSON data 

# V 1.0


# Installation

- Clone the git repo
```bash
git clone https://github.com/zhouzhuojie/Timbr_V1.git
cd timber
./install.sh
./watch.sh
```

# Run Website

- Change directory to project root
```bash
meteor
```
